## 卒業論文の概要

### 論文題目
**遥感衛星画像に基づく地表被覆分類システムの設計と実装**  
*Design and Implementation of a Land Cover Classification System Based on Remote Sensing Satellite Images*

### 研究内容
高解像度リモートセンシング画像の地表被覆分類における課題に対し、深層学習モデルを活用したWebシステムを開発した。

| 項目 | 詳細 |
|------|------|
| **システム構成** | B/Sアーキテクチャ、フロントエンドとバックエンドの分離設計 |
| **バックエンド** | Python FastAPI（非同期処理対応） |
| **フロントエンド** | Vue.js 3 + Element Plus |
| **深層学習モデル** | UNetMamba + 座標アテンション（CA）機構 → UNetMamba_CA |
| **使用データセット** | LoveDA、Vaihingen、Potsdam |
| **主要評価指標** | mIoU（平均IoU）、F1スコア |

### 主な成果
1. **モデルの改良**: UNetMambaの入力段に座標アテンション（CA）機構を追加し、性能を向上
2. **完全なシステム開発**: 画像アップロード、モデル選択、分割結果の可視化、AIアシスタント機能
3. **優れた汎化性能**: 複数のデータセットで安定した性能を確認
