# 每日学习工作流

> **原则**: 原理 → 操作 → 记录 → 反思

---

## 📋 每日流程模板

### 1. 开始前 (15 min)

```
[ ] 阅读当天任务清单 (task_checklist.md)
[ ] 阅读相关算法原理 (algorithm_principles.md)
[ ] 明确今天的目标产出
```

### 2. 执行中 (2-3 hours)

```
[ ] 按任务清单顺序操作
[ ] 遇到问题先自己思考 5 分钟
[ ] 及时截图（按命名规范）
[ ] 记录关键数据和观察
```

### 3. 结束后 (15 min)

```
[ ] 更新任务清单（打勾）
[ ] 保存截图到 02_Screenshots/
[ ] 写简短日记（今天学到什么）
```

## 🧠 学习心法

1. **不懂就标记**: 遇到不懂的概念，用 `[?]` 标记，之后查资料
2. **截图要解释**: 每张截图配一句话说明
3. **连接已知**: 每个新概念尝试和已知概念关联
4. **输出检验**: 能向别人解释清楚才算学会
